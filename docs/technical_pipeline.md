# Technical Pipeline

本系统将“未来热门学科专业趋势”拆成四个层次：新闻情境、画像化智能体、专业评分和群体聚合。

## 1. 新闻情境

历史期新闻由外部事实新闻 CSV/JSONL 提供。未来期新闻可以由 `job_sim.py` 中的新闻演化流程生成，也可以由使用者准备的外部情境数据替代。正式专业调查脚本会按照当前期和前若干半年期构造滚动新闻窗口，并将新闻按来源半年期写入 prompt。

## 2. 画像化智能体

profile JSONL 中的每个个体被视为一个模拟受访者。脚本会把省份、年龄、性别、教育水平、消费水平和发言摘录合成为受访者画像，让同一新闻环境下的不同个体给出差异化判断。

## 3. 专业评分

`scripts/run_major_outlook_survey_by_group.py` 是当前推荐的正式调查入口。它会：

1. 读取 `major/major_categories.md` 中的 99 个专业大类。
2. 按门类组织专业组，必要时拆分过大的门类。
3. 对每个 `时期 × profile × 专业组` 调用 LLM。
4. 要求 LLM 输出每个专业的就业前景评分、薪酬前景评分和简短理由。
5. 将结果解析为 `parsed_ratings.csv`。

若启用 `--score-history-periods`，当前期 prompt 会包含前若干期同专业组的历史统计均分，使模拟具有时间连续性。

## 4. 聚合与绘图

`scripts/aggregate_major_outlook_trends.py` 将 profile 级评分聚合为专业逐期均分、排名和趋势摘要。绘图脚本再生成专业级、门类级、细分群组级趋势图。

这些聚合表和图表是实验结果，本 code-only 包不包含它们。
